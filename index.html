Key Git Commands

  • Before you start working: git pull
  • Unsure what you changed? git status
  • To add/stage your work for commit: git add myfile.html or git add * to add everything you've changed
  • To stage your work ready for committing: git commit -m "my special message"
  • Then push the darn thang! git push origin master

    • Save the Day Git Commands

  • Shit! I did some stuff to my local file and I want to revert: git checkout -- myfile.html
  • I want to replace EVERYTHING I've just done locally: git fetch origin
